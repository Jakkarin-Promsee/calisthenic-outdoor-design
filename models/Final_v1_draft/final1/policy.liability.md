## 1. วัตถุประสงค์

เพื่อกำหนดให้ชัดเจนว่า ในโครงการออกแบบและก่อสร้างสนามคาลิสเทนิกส์กลางแจ้งนี้ **ผู้ออกแบบฟังก์ชันการใช้งานรับผิดชอบเฉพาะส่วนการออกแบบเชิงการใช้งานเท่านั้น** ส่วนความแข็งแรงของโครงสร้าง ความปลอดภัยทางวิศวกรรม งานเชื่อม ฐานราก การติดตั้ง และอายุการใช้งาน เป็นความรับผิดชอบของวิศวกรผู้ได้รับใบอนุญาตและผู้รับเหมาก่อสร้าง

## 2. ผู้เกี่ยวข้อง (คู่สัญญา)

| ฝ่าย | บทบาท |
|---|---|
| **ฝ่าย ก — ผู้ออกแบบฟังก์ชัน** (ผู้จัดทำเอกสารชุดนี้) | ออกแบบแนวคิด ผังการวาง และข้อกำหนดการใช้งาน · ประสานงาน · ตรวจการใช้งานเชิงคาลิสเทนิกส์ |
| **ฝ่าย ข — เจ้าของงาน/ผู้ว่าจ้าง** | เจ้าของพื้นที่และงบประมาณ ผู้ตัดสินใจว่าจ้าง และผู้ใช้งาน |
| **ฝ่าย ค — วิศวกรผู้ได้รับใบอนุญาต และผู้รับเหมาก่อสร้าง** | คำนวณและรับรองโครงสร้าง เลือกวัสดุ/เกรดเหล็ก งานเชื่อม ฐานราก ติดตั้ง และควบคุมงาน |

> ฝ่าย ค อาจเป็นเจ้าของงานเอง (ฝ่าย ข) ได้ หากเจ้าของงานมีใบอนุญาตประกอบวิชาชีพวิศวกรรมควบคุมสาขาที่เกี่ยวข้อง มิฉะนั้นต้องว่าจ้างวิศวกรภายนอก

## 3. ขอบเขตที่ฝ่าย ก (ผู้ออกแบบฟังก์ชัน) **รับผิดชอบ**

| รายการ | คำอธิบาย |
|---|---|
| **Concept Design** | แนวคิดและรูปแบบอุปกรณ์โดยรวม |
| **Layout** | ผังการวางอุปกรณ์ในพื้นที่ ระยะห่าง และทิศทางการใช้งาน |
| **Functional Requirement** | ข้อกำหนดเชิงการใช้งาน เช่น ความสูงบาร์ ความกว้างช่องจับ ระยะที่เหมาะกับแต่ละท่า |
| **User Requirement** | ความต้องการของผู้ใช้ (ผู้ใหญ่ + เด็ก) และความเหมาะสมกับท่าออกกำลังกาย |
| **การประสานงาน** | ประสานระหว่างเจ้าของงาน วิศวกร และผู้รับเหมา รวบรวมข้อมูลให้ครบ |
| **การตรวจการใช้งานเชิงคาลิสเทนิกส์** | ตรวจว่าเมื่อสร้างเสร็จ ระยะและรูปแบบใช้เล่นคาลิสเทนิกส์ได้จริงตามที่ออกแบบ |

> เอกสารแบบและการคำนวณเบื้องต้นที่ฝ่าย ก จัดทำ (เอกสาร 1/3, 2/3, 3/3) เป็นข้อมูลประกอบเพื่ออำนวยความสะดวกให้ฝ่าย ค นำไปคำนวณและรับรองต่อ มิใช่แบบที่รับรองแล้ว

## 5. ข้อจำกัดความรับผิด (Liability Disclaimer)

1. แบบและการคำนวณที่ฝ่าย ก จัดทำเป็นการ **ออกแบบเชิงฟังก์ชันและประเมินเบื้องต้นจากการค้นคว้าเอกสารมาตรฐานและข้อมูลผู้ผลิต** มิใช่การคำนวณโครงสร้างที่วิศวกรลงนามรับรอง
2. ฝ่าย ก **ไม่รับประกัน** ความแข็งแรง ความปลอดภัย หรือความเหมาะสมเชิงโครงสร้างของอุปกรณ์
3. ฝ่าย ก **ไม่รับผิด** ต่อความเสียหาย การบาดเจ็บ หรือการสูญเสียใด ๆ อันเกิดจากการก่อสร้าง การวิบัติของโครงสร้าง งานเชื่อม ฐานราก การติดตั้ง คุณภาพวัสดุ หรือการใช้งานที่เกินกว่าที่ออกแบบ
4. การนำแบบไปก่อสร้างถือว่าฝ่าย ข รับทราบและยอมรับว่า **ต้องว่าจ้างวิศวกรผู้ได้รับใบอนุญาตตรวจสอบ คำนวณ และรับรองก่อนดำเนินการ**
5. หากมีการแก้ไขแบบหรือวัสดุจากที่ระบุไว้ ฝ่าย ก ไม่รับผิดชอบต่อผลที่ตามมา และการแก้ไขต้องผ่านการตรวจสอบโดยวิศวกร

## 6. นโยบายการทำงาน (Working Policy)

1. ฝ่าย ก **จะไม่ลงนามในฐานะ "ผู้รับเหมาก่อสร้าง"** และจะลงนามได้เฉพาะในฐานะ "ผู้ออกแบบฟังก์ชันการใช้งานและผู้ประสานงาน" เท่านั้น
2. ก่อนเริ่มก่อสร้าง ฝ่าย ข ต้องจัดให้มี **วิศวกรผู้ได้รับใบอนุญาต** ตรวจสอบและรับรองแบบ และจัดให้มีผู้ควบคุมงานตามที่กฎหมายกำหนด
3. การก่อสร้างจะเริ่มได้ต่อเมื่อแบบผ่านการรับรองโดยวิศวกรแล้วเท่านั้น
4. แบ่งความรับผิดชอบเป็นลายลักษณ์อักษร ลงนามครบทั้ง 3 ฝ่าย (ตามตารางข้อ 2) และเก็บสำเนาไว้ทุกฝ่าย
5. หากฝ่าย ข เลือกใช้ช่างที่อ่านแบบวิศวกรรมไม่ได้หรือไม่มีวิศวกรรับรอง ฝ่าย ก จะทำหน้าที่เพียงตรวจระยะการใช้งานเชิงคาลิสเทนิกส์ และ **ไม่รับผิดชอบความแข็งแรงของโครงสร้าง**

## 7. ข้อกฎหมายที่เกี่ยวข้อง (อ้างอิงเบื้องต้น — ต้องให้นักกฎหมายยืนยัน)

> รายการนี้รวบรวมเพื่อให้เห็นภาพความเสี่ยง **มิใช่การยืนยันทางกฎหมาย** ต้องให้ทนายความ/นิติกรตรวจสอบความถูกต้องและการบังคับใช้กับกรณีนี้

| กฎหมาย | สาระที่เกี่ยวข้อง (โดยสังเขป) |
|---|---|
| **พ.ร.บ. วิศวกร พ.ศ. 2542** | งานออกแบบ/คำนวณโครงสร้างรับน้ำหนักอาจเข้าข่าย "งานวิศวกรรมควบคุม" ที่ต้องทำโดยผู้ได้รับใบอนุญาตจากสภาวิศวกร |
| **พ.ร.บ. ควบคุมอาคาร พ.ศ. 2522** | สิ่งปลูกสร้าง/โครงสร้างบางประเภทต้องขออนุญาตและมีวิศวกรลงนามแบบ — ต้องตรวจสอบกับองค์กรปกครองส่วนท้องถิ่นว่ากรณีนี้เข้าข่ายหรือไม่ |
| **ประมวลกฎหมายอาญา ม. 291 / ม. 300** | การกระทำโดยประมาทเป็นเหตุให้ผู้อื่นถึงแก่ความตาย / ได้รับอันตรายสาหัส มีโทษทางอาญา |
| **ประมวลกฎหมายแพ่งและพาณิชย์ ม. 420 (ละเมิด)** | ผู้กระทำโดยจงใจหรือประมาทเลินเล่อทำให้ผู้อื่นเสียหาย ต้องชดใช้ค่าสินไหมทดแทน |

> เน้นย้ำ: ความรับผิดทางอาญาและความรับผิดต่อผู้ใช้งานที่บาดเจ็บ มักไม่สามารถยกเว้นได้ด้วยสัญญาส่วนตัว — การลดความเสี่ยงที่ได้ผลคือการไม่รับบทบาทเกินความสามารถ และให้วิศวกรผู้มีใบอนุญาตรับรองงานจริง
