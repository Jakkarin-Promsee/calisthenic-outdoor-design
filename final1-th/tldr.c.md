# สรุปย่อสำหรับเจ้าของงาน/วิศวกรผู้รับรอง (ฝ่าย ค)

## TL;DR ฉบับอ่านเร็ว · สนามคาลิสเทนิกส์กลางแจ้ง

| หัวข้อ              | รายละเอียด                                                                                                   |
| ------------------- | ------------------------------------------------------------------------------------------------------------ |
| เอกสารนี้คือ        | สรุปย่อให้อ่านได้ใน 2–3 นาที ว่าโครงการคืออะไร ตรวจอะไรไปแล้ว และมีจุดใดที่ขอความเห็นทางวิศวกรรมก่อนก่อสร้าง |
| สถานะ               | แบบเชิงการใช้งาน + การประเมินเบื้องต้นจากการค้นคว้า **(ยังมิใช่แบบที่ผ่านการรับรองทางวิศวกรรม)**             |
| ปรับปรุงล่าสุด      | 17 มิถุนายน 2569                                                                                             |
| ฉบับเต็มสำหรับลงนาม | [หนังสือส่งมอบและรับทราบบทบาท (policy.handoff.md)](policy.handoff.md)                                        |

> **คำชี้แจง** ข้อมูลและตัวเลขทั้งหมดรวบรวมจากแหล่งสาธารณะ (มาตรฐาน EN 16630/1176, Hilti, ผู้ผลิต) ส่งมอบในลักษณะ "ตามที่เป็น" เพื่อให้ท่านในฐานะวิศวกรผู้ได้รับใบอนุญาตพิจารณา ตรวจสอบ และรับรองด้วยดุลพินิจวิชาชีพก่อนดำเนินการ

---

## 1. โครงการโดยย่อ

สนามคาลิสเทนิกส์กลางแจ้งหลังบ้าน · พื้นที่ ~6×5 ม. · ผู้ใช้ ผู้ใหญ่ 1 + เด็ก ~5 ขวบ เหล็กชุบกัลวาไนซ์ทั้งหมด · อุปกรณ์ 4 ชิ้น + พื้นยาง:

- **Pull-up bar** (เสา SHS 76, บาร์ OD 34) · **Snake bar** (เสา SHS 76, ท่อ OD 42 + OD 34)
- **Dip bar** (โครง OD 42) · **Monkey + Flying bar** ยาว 5 ม. (เสา SHS 50, ซี่ OD 34) · **พื้นยาง EPDM**

---

## 2. สรุปผลตรวจ

ภายใต้โหลดมาตรฐาน **EN 16630:2015 (1,942 N/จุด รวมพลศาสตร์)** และวัสดุตามแบบ (ผนัง 3.2 มม.) การประเมินเบื้องต้น (first-order) พบว่า **ทุกชิ้นส่วนผ่าน ไม่มีชิ้นใดคราก** · โครงสร้างหลัก (เสา ฐานราก สลัก) มี margin สูง (utilization 0.2–0.5)

---

## 3. จุดที่ขอความเห็น/การพิจารณาของท่านเป็นพิเศษ

เรียงตามลำดับความสำคัญ (รายละเอียดเต็มใน [spec.summary §3](spec.summary.md)):

| จุด                                             | ประเด็นโดยย่อ                                                                                                  |
| ----------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **1. ค้ำยันสามเหลี่ยม Monkey (Zone 4)**         | เสาสูง 2.70 ม. ผ่านได้เพราะมีค้ำยันนี้ — แนะนำยืนยันการเชื่อมและห้ามใช้งานโครงโดยไม่มีค้ำยัน                   |
| **2. บาร์ดึงข้อสูง OD 34 ช่วง 1.5 ม.**          | margin ปานกลาง (util 0.71 ที่ผู้ใช้ 1 คน) — แนะนำเชื่อมปลายยึดแน่น และพิจารณาว่าจะใช้ทีละคนหรือเพิ่มความหนาท่อ |
| **3. ชิ้นยื่น (ราวจับข้าง snake / มือจับ dip)** | โมเมนต์งัดที่รอยเชื่อม (snake side grip util 0.95) — แนะนำเชื่อมเต็มรอบ หรือลดระยะยื่น                         |
| **4. พื้นยางกันการตก**                          | Monkey ตกได้สูง ~1.70 ม. เกินที่ยาง 25 มม. รองรับ — แนะนำเพิ่มความหนายางใต้ Monkey/Pull-up เป็น ≥ 50 มม.       |
| **5. ผิวกันสนิม + งานเชื่อม**                   | เลือกระหว่างชุบกัลวาไนซ์จุ่มร้อนทั้งชุด vs สเปรย์กัลวาไนซ์เย็น และยืนยันคุณภาพรอยเชื่อมจุดรับแรงพลศาสตร์       |

---

## 4. สิ่งที่อยู่ในขอบเขตการตัดสินใจของท่าน

ในฐานะเจ้าของงาน + วิศวกรผู้ได้รับใบอนุญาต + ผู้ตรวจงาน:

1. ตรวจสอบ/คำนวณยืนยัน และ **รับรองแบบ** ก่อนก่อสร้าง
2. ยืนยันฐานราก: สลัก M12×120 ในพื้นคอนกรีต C20/25 หนา ≥ 140 มม. (ตรวจความหนา/สภาพปูนหน้างานจริง)
3. ควบคุมคุณภาพหน้างาน (รอยเชื่อม ความลึกการฝังสลัก) และตรวจรับก่อนเปิดใช้งาน — **โดยเฉพาะก่อนให้เด็กใช้**

> แนวทางแก้ไขทั้งหมดเป็นเพียงข้อเสนอเพื่อพิจารณา · ระดับและวิธีการตรวจสอบ ตลอดจนการอนุมัติ เป็นดุลพินิจวิชาชีพและความรับผิดชอบของท่าน
