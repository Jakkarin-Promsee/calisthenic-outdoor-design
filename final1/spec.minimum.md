> # คำเตือนสำคัญ — อ่านก่อนใช้ทุกครั้ง
>
> **เอกสารนี้ไม่รับรองการใช้งานจริงโดยเด็ดขาด** ตัวเลข "ขั้นต่ำ" ทุกค่าในไฟล์นี้คือจุดที่วัสดุ **เริ่มคราก (yield) พอดีภายใต้โหลดออกแบบ** — กล่าวคือเป็นเส้น **"ต่ำกว่านี้พังแน่"** มิใช่เส้น "เท่านี้แล้วใช้ได้"
>
> - ค่าขั้นต่ำหลัก = util 1.0 (σ = fy = 235 MPa) → **เหลือ margin ความปลอดภัย = 0 (ศูนย์)**
> - **ไม่รวม:** ความล้าจากการใช้ซ้ำ (fatigue), คุณภาพรอยเชื่อม, การโก่งเดาะ (buckling), แรงลม, ความเผื่อสนิม, ความคลาดเคลื่อนหน้างาน — ปัจจัยเหล่านี้ทำให้ของจริงพังได้แม้วัสดุ "ผ่าน" เส้นขั้นต่ำนี้
> - **จุดประสงค์เดียว:** ใช้เป็น _ตัวกรองตัด choice_ ตอนเดินเลือกของจากร้านวัสดุ — ถ้าของที่ร้านมี **เล็กกว่า** ค่าขั้นต่ำ → ตัดทิ้งทันที ไม่ต้องเสียเวลา · ถ้า **โตกว่าหรือเท่ากับ** → เป็นเพียง _ตัวเลือกที่ยังไม่ตกรอบ_ ต้องส่งให้ผู้รับรอง (ฝ่าย ค) ตรวจและคำนวณยืนยันต่อ
> - **สเปกที่ใช้สร้างจริงคือ [spec.full.md](spec.full.md) เท่านั้น** (เหล็กหนา 3.2 มม. ซึ่งคงระยะห่างจากจุดครากไว้) — ไฟล์นี้มิใช่ของแทน และ **ห้ามนำไปก่อสร้าง สั่งของตามลำพัง หรืออ้างเป็นข้อกำหนด**
> - **ต้องคำนวณใหม่ทั้งไฟล์ทุกครั้งที่มีการแก้วัสดุจริง** (เปลี่ยนเกรดเหล็ก fy, เปลี่ยน OD/หน้าตัด, เปลี่ยนช่วงพาด, เปลี่ยนโหลดออกแบบ) — ค่าทุกค่าผูกกับสมมติฐานในข้อ 2 ถ้าสมมติฐานเปลี่ยน ตัวเลขทั้งหมดเป็นโมฆะ

## 1. ไฟล์นี้คืออะไร / ไม่ใช่อะไร

| ไฟล์นี้ **เป็น**                                                 | ไฟล์นี้ **ไม่เป็น**                              |
| ---------------------------------------------------------------- | ------------------------------------------------ |
| ตัวช่วยกะว่า "วัสดุที่หาได้จากร้าน เล็กเกินไปจนตัดทิ้งได้เลยไหม" | สเปกสำหรับสร้างจริง                              |
| เส้นพื้น (floor) เชิงทฤษฎีว่าลดวัสดุได้มากสุดแค่ไหนก่อนคราก      | คำรับรองว่า "เกินเส้นนี้แล้วปลอดภัย/ใช้ได้ 100%" |
| เครื่องมือเทียบ "สเปกเต็ม (3.2 มม.) เผื่อเกินขั้นต่ำไว้เท่าไร"   | การคำนวณโครงสร้างฉบับวิศวกรลงนาม                 |
| ตัวเลขที่ผูกกับสมมติฐานคงที่ชุดหนึ่ง (ข้อ 2)                     | ค่าที่ใช้ได้ถาวรโดยไม่ต้องคำนวณซ้ำ               |

**กฎการใช้ (decision rule):**

> สำหรับแต่ละชิ้น เทียบ **section modulus (Z) ของวัสดุที่ร้านมี** กับค่า **Z ขั้นต่ำ** ในตาราง
>
> - `Z(ของร้าน) < Z_min` → **ตัดทิ้งทันที** (ของจริงจะครากภายใต้โหลดออกแบบ)
> - `Z(ของร้าน) ≥ Z_min` → **ยังไม่ตกรอบ** — เก็บไว้เป็นตัวเลือก ส่งให้ฝ่าย ค ตรวจรับรอง **ห้ามถือว่าผ่านเอง**
> - เทียบกับคอลัมน์ **"ขั้นต่ำคง margin (0.6fy)"** ด้วย: ถ้าของร้านถึงเส้นนี้ = มี margin ใกล้เคียงสเปกออกแบบเดิม (อุ่นใจกว่ามาก) · ถ้าอยู่ระหว่างสองเส้น = ใช้ได้เชิงกำลังแต่ margin บางลง ต้องให้ผู้รับรองชี้ขาด

## 2. วิธีคำนวณและสมมติฐาน (ฐานของทุกตัวเลข — แก้เมื่อไร คำนวณใหม่หมด)

### 2.1 สมมติฐานคงที่

- **โหลดออกแบบ P = 1,942 N** ต่อจุดใช้งาน 1 คน (EN 16630 Table 1 รวม Cdyn = 2.0) — เท่ากับเอกสาร 3/4 และ 4/4
- **เหล็ก fy = 235 MPa** (เหล็กกล่อง/ท่อกัลวาไนซ์ทั่วไป ค่าอนุรักษ์)
- ท่อจับ/ซี่เชื่อมปลายทั้งสองด้าน = สภาพ **ปลายยึดแน่น** (โมเมนต์กลางช่วง = P·L/8) · ชิ้นยื่น = **คานยื่น** (โมเมนต์ที่โคน = P·a)
- หน้าตัดท่อจับ **OD คงที่** (34 มม. / 42 มม.) ตามหลักการจับของ EN 16630 (16–45 มม.) — ลดได้เฉพาะ **ความหนาผนัง (wall)** เท่านั้น ห้ามลด OD
- น้ำหนักตัวเอง (self-weight) ละไว้ (โหลด live 1,942 N ครอบงำ ช่วงสั้น) — เป็นการละแบบอนุรักษ์เล็กน้อย

### 2.2 โมเมนต์ที่ต้องรับ (demand) — **ไม่ขึ้นกับวัสดุ จึงเป็นค่าตั้งต้นคงที่**

### 2.3 สูตรหา "ขั้นต่ำ"

1. **section modulus ขั้นต่ำ:**
   - เส้นคราก (หลัก): `Z_min = M / fy = M / 235`
   - เส้นคง margin (อ้างอิง): `Z_min(0.6fy) = M / (0.6·fy) = M / 141`
     _(M เป็น N·mm, Z_min เป็น mm³)_
2. **section modulus ของหน้าตัดที่จะเลือก:**
   - ท่อกลม: `Z = (π/32)·(D⁴ − (D−2t)⁴) / D`
   - กล่อง SHS: `Z = (B⁴ − (B−2t)⁴) / (6B)`
3. **หาผนังขั้นต่ำ t (เมื่อ OD/B คงที่):**
   `t ≥ (D − ⁴√(D⁴ − 10.186·Z_min·D)) / 2` (10.186 = 32/π)
   แล้ว **ปัดผนังขึ้น** หาขนาดที่ร้านมีจริง (สต๊อกผนังทั่วไป: 1.5 / 2.0 / 2.3 / 2.5 / 2.8 / 3.0 / 3.2 / 3.5 / 4.0 มม.)

### 2.4 เกณฑ์อ่านค่า util (= σ/fy)

util ≤ 0.6 = สเปกออกแบบเผื่อ margin · 0.6–1.0 = ยังไม่คราก แต่ margin ลดลงเรื่อย ๆ · **= 1.0 = ครากพอดี (เส้นขั้นต่ำหลักของไฟล์นี้)** · > 1.0 = คราก/พัง

### 3.6 เสา POST — ถ้าคง OD/SHS เดิม ลด wall ได้สูงสุดเท่าไหร่

> **บริบทที่เพิ่มหัวข้อนี้:** wall หนาหายากจากร้านทั่วไป — ยิ่งต้องการ wall มาก ยิ่งหาสต๊อกยาก เป้าหมายคือรู้ว่า "ถ้าคงขนาด SHS เดิม (ห้ามย่อ OD/B) ลด wall ลงได้ต่ำสุดถึง wall เท่าไหร่ก่อนเส้นคราก" — ใช้ตัดสินใจตอนเดินดูสต๊อกร้าน
>
> **ข้อจำกัดเช่นเดิม:** ตัวเลขนี้คิดเฉพาะ **การดัดที่โคนเสา (bending demand)** เท่านั้น เสาจริงรับโหมดรวม (อัด + ดัด + โก่งเดาะ + สลักฐาน) — ค่าที่ได้คือ **เพดานทางทฤษฎี** ห้ามใช้ลดจริงโดยไม่ผ่านผู้รับรอง

#### สูตรที่ใช้ (SHS กล่องสี่เหลี่ยม)

`Z = (B⁴ − (B−2t)⁴) / (6B)`

หา t ขั้นต่ำ: `(B−2t)⁴ ≥ B⁴ − 6B·Z_min` → `t_min = (B − ⁴√(B⁴ − 6B·Z_min)) / 2`

แล้วปัดขึ้นหาสต๊อกที่มีจริง (สต๊อกผนัง SHS ที่หาได้บ่อย: 1.5 / 2.0 / 2.3 / 2.5 / 2.8 / 3.0 / 3.2 / 3.5 / 4.0 มม.)

#### ผลคำนวณ — เสา SHS 76 (pull-up ×3 + snake ×2)

| รายการ              | M (kN·m) | Z_min คราก (mm³) | Z ของ SHS 76 × wall ต่าง ๆ                                            | wall ขั้นต่ำ (คราก)           | **สเปกจริง**   | util สเปกจริง |
| ------------------- | -------: | ---------------: | --------------------------------------------------------------------- | ----------------------------- | -------------- | ------------: |
| เสา pull-up / snake |     2.13 |            9,064 | 76×1.5 → Z≈10,887 ✓ · 76×1.4 → Z≈10,205 ✓ · ต่ำกว่า 1.4 → Z < 9,064 ✗ | **1.5 มม.** (ปัดขึ้นจาก ~1.4) | SHS 76×**3.2** |          0.42 |

- **SHS 76 คงขนาด เชิงดัดล้วน** ลด wall ลงได้ถึง **1.5 มม.** (สต๊อกต่ำสุดที่มีจริงที่ยังไม่คราก) — ห่างจากสเปกจริง 3.2 มม. มาก เพราะ SHS 76 มีหน้าตัดใหญ่เผื่อการดัดไว้มาก
- **แต่ห้ามลดจริง:** เสา SHS 76 รับ _แรงอัด + โก่งเดาะ + โมเมนต์ฐาน_ ร่วมกัน wall บางทำให้ผนังแบนบุบง่าย ความชะลูด (slenderness = L/r) เพิ่ม ซึ่งลดกำลังโก่งเดาะอย่างมีนัยสำคัญ ค่า 1.5 มม. นี้จึงเป็น **เส้นกรองตัดทิ้งเฉพาะกรณีร้านเสนอสต๊อก SHS 76 wall น้อยกว่า 1.5 เท่านั้น** (ซึ่งในทางปฏิบัติไม่ค่อยเจอ)

#### ผลคำนวณ — เสา SHS 50 (monkey bar)

| ------------------------- | -------: | ---------------: | --------------------------------------- | -------------------------------- | -------------- | ------------: |

| เสาสูง — **มีค้ำยัน**     |     0.65 |            2,766 | 50×1.0 → Z≈2,867 ✓ · 50×0.8 → Z≈2,540 ✗ | **1.0 มม.\*** (ปัดขึ้นจาก ~0.85) | SHS 50×**2.8** |          0.35 |

## 6. เมื่อแก้วัสดุจริง — ทำซ้ำอย่างไร

1. อัปเดต **สมมติฐานข้อ 2.1** ที่เปลี่ยน (เช่น fy ของเหล็กเกรดใหม่, OD ใหม่, ช่วงพาดใหม่, โหลด P ใหม่)
2. ถ้าเรขาคณิต/โหลดเปลี่ยน → คำนวณ **demand M ใหม่** (ตาราง 2.2) · ถ้าเปลี่ยนแค่วัสดุ → M เดิมใช้ได้
3. คำนวณ `Z_min = M/fy` และ `Z_min(0.6fy) = M/141` ใหม่ (ข้อ 2.3)
4. แทนหน้าตัด/ผนังที่ร้านมีลงสูตร Z แล้วเทียบกับ Z_min ตาม **กฎการใช้ (ข้อ 1)**
5. อัปเดตวันที่หัวเอกสาร + บันทึกว่าแก้อะไร
6. **ควรส่งผลลัพธ์ให้ฝ่าย ค ตรวจรับรองก่อนสั่งของหรือสร้างทุกครั้ง** — ไฟล์นี้ไม่แทนการรับรอง
