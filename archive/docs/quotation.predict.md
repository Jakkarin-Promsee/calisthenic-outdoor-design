## หมายเหตุทั่วไป

- **ราคาประมาณการ** อิงจากราคาตลาด มิ.ย. 2569 — ควรขอใบเสนอราคาจากร้านจริงก่อนสั่งซื้อ
- **galvanized ทั้งหมด** → ไม่มีค่าพ่นสี / ค่ากันสนิมเพิ่ม
- **PIECE 3 (Dip Bar)** ไม่มีเสา — frame แต่ละตัวยืนอิสระ ยึดพื้นที่ y=0 และ y=0.75m ของ frame นั้น
- **PIECE 4 (Monkey Bar)** ต้องวางบน concrete pad ก่อนติดตั้ง — anchor bolt ฝังในคอนกรีต
- สั่งเหล็ก SHS / ท่อกลม galv จากร้านเหล็กทั่วไป เช่น SB Steel, Aplus Watsadu, Wongguru
- ค่าดัดท่อ: หาร้านรับดัดท่อกลม OD 34/42mm ที่มีเครื่อง rotary-draw bender (ไม่ใช่แบบ press bend)
- **งบส่วนต่าง ~60,000 THB** รองรับ: contingency 10–15%, อัปเกรด EPDM หนาขึ้น, ไฟ LED กลางแจ้ง, หรือเพิ่มอุปกรณ์อื่น
