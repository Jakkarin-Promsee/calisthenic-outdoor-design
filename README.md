# 🏋️ Calisthenic Gym Design — สนามคาลิสเทนิกส์กลางแจ้ง

โปรเจกต์ออกแบบ **สนามออกกำลังกายคาลิสเทนิกส์กลางแจ้ง** สำหรับบ้านพักอาศัย
ผู้ใช้: ผู้ใหญ่ + เด็ก (~5 ขวบ) · งบประมาณ ~150,000 บาท · พื้นที่ ~6×5 ม.
โมเดล 3 มิติทั้งหมดสร้างใน **Blender** ผ่าน BlenderMCP

---

## 👉 เริ่มอ่านที่นี่

**ชุดเอกสารส่งมอบฉบับจริง (source of truth) อยู่ในโฟลเดอร์ [`final1/`](final1/) — เปิดสารบัญรวมที่ [`final1/toc.md`](final1/toc.md)**

`toc.md` คือดัชนีนำทางที่ลิงก์ไปทุกเอกสาร (แบบและข้อกำหนด · การวิเคราะห์การรับแรง · ฐานราก · ขอบเขตความรับผิด · งานค้นคว้าอ้างอิง) พร้อม **ลำดับการอ่านที่แนะนำ**

> **เริ่มเร็ว** — ฝ่ายประสานงาน: [`final1/tldr.b.md`](final1/tldr.b.md) · เจ้าของงาน/วิศวกร: [`final1/tldr.c.md`](final1/tldr.c.md)

## 📁 โครงสร้างโฟลเดอร์

| โฟลเดอร์ | คืออะไร |
|---|---|
| **[`final1/`](final1/)** | ★ **ชุดเอกสารส่งมอบ — source of truth** (formal ไทยทางการ) · เริ่มที่ [`toc.md`](final1/toc.md) |
| [`archive/`](archive/) | เอกสารรุ่นเก่า/ฉบับก่อนที่ `final1/` แทนที่แล้ว — **เก็บไว้อ้างอิงเท่านั้น ห้ามใช้สร้างจริง** ([รายละเอียด](archive/README.md)) |
| `models/` | ไฟล์ Blender `.blend` ทุกชิ้น (ชุดปัจจุบันอยู่ใน `models/Final_v1_draft/`) |
