# 🏋️ Calisthenic Gym Design — สนามออกกำลังกายกลางแจ้ง

โปรเจกต์ออกแบบ **สนามคาลิสเทนิกส์กลางแจ้ง** สำหรับบ้านพักอาศัย
ใช้สำหรับผู้ใหญ่ + เด็ก (~5 ขวบ) · งบประมาณ ~150,000 บาท · พื้นที่ ~6×5 ม.

โมเดล 3 มิติทั้งหมดสร้างใน **Blender 4.x** ผ่าน BlenderMCP

---

## 📁 โครงสร้างโฟลเดอร์

| โฟลเดอร์ | เก็บอะไร |
|---|---|
| `models/` | ไฟล์ `.blend` ทุกชิ้น — เปิดแก้โมเดลที่นี่ (`_backups/` = auto-backup) |
| `renders/final/` | รูป render ตัวจริงที่ใช้นำเสนอ |
| `renders/drafts/` | รูประหว่างทาง / ทดลอง / sketch |
| `renders/animation/` | วิดีโอเดินชม + พาโนรามา + gif (`_frames/` = เฟรมดิบ ~2.6 GB ลบได้) |
| `docs/` | `spec.md` (สเปกหลัก), `measurements.md` (ขนาดวัดจริง), `quotation.md` (ใบเสนอราคา) |
| `assets/` | ไฟล์ HDRI (แสงสภาพแวดล้อม) |
| `scripts/` | สคริปต์ Python สำหรับ render/gif |

---

## 🧱 อุปกรณ์ในสนาม (5 ชิ้น)

| ชิ้นงาน | ความสูง | จุดเด่น |
|---|---|---|
| Pull-up bar (บาร์โหน) | เสา 2.20 ม. / บาร์ 2.10 ม. | บาร์จับ 1.5 ม., เสา SHS 100 มม. |
| Snake bar (บาร์งู) | เสา 2.00 ม. / บาร์ 1.90 ม. | กว้าง 2 ม., คลื่น 4 ลูก ลึก 50 ซม. |
| Dip bar (บาร์ดิป) | 1.20 ม. | 3 เฟรม, กว้าง 60 + 50 ซม. |
| Monkey + Flying bar (บาร์ลิง) | 0.5–2.6 ม. (เอียง 35°) | ยาว 4 ม. กว้าง 1.0 ม., 4 โซน + บันไดข้าง |
| Floor (พื้น) | หนา 0.30 ม. | สลैब 6×5 ม., โซนวางอุปกรณ์ 4×3 ม. |

> สเปกละเอียดทุกชิ้น → ดู [`docs/spec.md`](docs/spec.md)

---

## 👀 วิธีเปิดดูงาน

1. เปิด `models/main_scene.blend` ใน Blender → เห็นสนามรวมทุกชิ้น
2. ดูภาพนิ่งล่าสุด → `renders/final/`
3. ดูวิดีโอเดินชม → `renders/animation/walkthrough.mp4`
